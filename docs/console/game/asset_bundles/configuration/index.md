## Create configuration

1. Click on **Create configuration**
2. A Create configuration modal will appear with the following fields:
    - **Asset bundle** Name of the new Asset Bundle
    - **Start date** Start date for asset bundle
    - **End date** End date for asset bundle
    - **Is Localtime** If it's localtime
    - **Segment data** Selected segments
    - **Platforms** Available for these platforms per version range
3. Click on **Save** to store the changes and optionally select the **Create another** checkbox in order to create another creative


## Edit configuration
1. Click on **Edit configuration** ![pencil](https://github.com/azerion/gamedock-sdk/raw/master/docs/console/_images/pencil.png) button on the selected row
2. An Edit configuration modal will appear with the following fields:
    - **Asset bundle** Name of the new Asset Bundle
    - **Start date** Start date for asset bundle
    - **End date** End date for asset bundle
    - **Is Localtime** If it's localtime
    - **Segment data** Selected segments
    - **Platforms** Available for these platforms per version range
3. Click on **Save** to store the changes

## Remove configuration
1. Click on **Remove configuration** ![trash](https://github.com/azerion/gamedock-sdk/raw/master/docs/console/_images/trash.png) button 
2. You will be prompted to verify your action or with a reject message detailing the reasons for the rejection.
   This action will delete the configuration.
